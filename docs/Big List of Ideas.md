*Ideas for Classify, in no particular order*

Random ideas I think of go here until I can work them into the roadmap. They can sit here instead of being forced into the near-future work so I don't run into feature creep.

* **File Trimming** Use AI or heuristic algorithms to identify the start and end of music in files in order to trim the silence and non-music before and after a performance
* **Biographies & Descriptions** Search Wikipedia or other sources for bios on composers and/or descriptions for pieces.
* **API-first Matching** Search APIs like MusicBrainz for info about audio files in order to try and generate matches without user input. 
* **Local AI Matching** Use a local AI setup (Ollama, Gemini, etc.) to locate matches when APIs don't work. Generate fingerprints and adjectives plus additional data (duration, file name, all metadata) to help the AI.
